# Game Controls
[MouseMovement] - `Look`

[RightMouse] - `Scope` (only works with Sniper)

[Esc] - Exit Armory Player (only works in Krom Player)

[W] - Walk `Forward`

[S] - Walk `Backward`

[A] - Walk `Left`

[D] - Walk `Right`

[1] - Switch to `Pistol`

[2] - Switch to `Assault`

[3] - Switch to `Ak47`

[4] - Switch to `Sniper`

[5] - Switch to `Shotgun`
